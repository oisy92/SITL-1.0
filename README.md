# Autopilot Failure Mode Analysis using Simulation in the Loop

This is a modified implementation of Ardupilot ArduCopter to create an autonomous UAV flight simulator for autopilot studies and analysis.

## Prerequisites
1. C++ enviornment
2. Python
3. MATLAB
4. Linux Machine
